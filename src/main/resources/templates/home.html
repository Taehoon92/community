Welcome to My Community

This website is the personal side project using Spring boot. Common community site features are implemented.

Key features implemented

* Post CRUD APIs. (Create, Read, Update and Delete)

* File Upload

* Hierarchical structured comment

* Sign in & Sign up APIs. (Using JWT and Spring Security)

* Social Login - OAuth2

* Setting user's role and authorization according to the role

* Parameter validation on sign in, sign up and post creation

* Admin page - role modify, user delete (Admin role only)

* User page - check own posting, change password, delete account

* Swagger API documentation provided

* Internationalize and message (English and Korean - Depends on web browser language setting)
